# Plugin Prompt Cache

This directory stores AI-facing usage notes for plugins that are:

- important
- easy to misuse
- subtle in contract
- environment-sensitive
- or repeatedly encountered during development

## Working Rule

When development uncovers reusable plugin understanding:

- add a short note here
- keep it operational
- prefer one file per plugin or plugin family

Do not document every plugin by default.

The plugin code and `meta` remain the source of truth.
This directory exists to cache learned usage patterns that help AI use plugins
correctly in future tasks.
